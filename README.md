# 全马破4训练计划（Marathon Sub-4 Planner）

一个纯前端、单文件的训练计划清单页面，用于记录赛前训练安排与每日执行状态。

## 功能

- 按周展示训练计划（跑步、休息、足球、比赛日等）
- 顶部进度条显示完成数量与百分比
- 本地持久化（刷新页面后状态不丢失）
- 完成交互带动画反馈（撒花 + 振动，视浏览器支持）

## 交互说明

- **单击任务卡片**：标记完成/取消完成  
  - 标记完成时会触发撒花动画
- **双击任务卡片**：标记跳过/取消跳过  
  - 跳过状态显示为 `[-] SKIPPED`
  - 标记跳过时会弹出输入框，填写跳过原因
- **长按任务卡片（约 450ms）**：填写训练备注（不影响完成/跳过状态）
- **右键任务卡片（桌面端）**：同样可打开训练备注输入框

## 数据存储

- 使用浏览器 `localStorage`
- 存储键：`marathon_tasks_v2`
- 存储内容包含：
  - `done`（是否完成）
  - `skipped`（是否跳过）
  - `skipNote`（跳过原因）
  - `extraNote`（训练备注）

## 运行方式

这是一个静态页面项目，无需构建：

1. 直接用浏览器打开 `index.html`
2. 或用任意静态服务器托管后访问

## 文件结构

- `index.html`：页面结构、样式、训练数据和交互逻辑
